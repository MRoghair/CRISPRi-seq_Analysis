# CRISPRi-seq_Analysis
Code used to analyze CRISPRi-seq data, updated November 2025

The file `Library_for_MAGeCK_v3.txt` is a key that contains all sgRNAs in the library, which is necessary for the MAGeCK program to analyze the data. The file `20251030_MRS_CRISPRiSeqDataAnalysis_V1.md` contains all data analysis scripts, updated November 2025.
